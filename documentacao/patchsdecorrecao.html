Instalação Patchs de correção.

Baixe o patch que deseja aqui, você pode ver todos os fontes em nosso Github
Navegue até a pasta onde se localiza os Downloads, em nosso caso fica em /home/NomeDoUsuario/Downloads.

$ cd /home/slackjeff/Downloads

Feito isto vamos agora entrar como root

$ su

Agora vamos instalar o patch rodando dpkg -i nomedopacote.deb

# dpkg -i patchversao.deb

Pronto, agora podemos remover o pacote .deb

# rm -v patchversao.deb
